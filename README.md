# theLab Boozang

## Description

Frontend platform built with React.js for learning test automation and the Boozang tool. Mock backend with json-server. Version 2.

## Installation

To install

`npm install`

To run

`npm start`

`npm run server`

## To build

`npm run build`

Then use the dist folder you get.

## Built With

- React.js
- Scss
- Bootstrap 4
